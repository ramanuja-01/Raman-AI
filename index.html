🧬 1. ENSEMBLE CLASSIFIER MATHEMATICS [TAP TO EXPAND]

The symptom classifier uses an L2-normalized TF-IDF vector space running across a triple-model ensemble:

  • Filtered Unigrams & N-Grams: Strips punctuation, filters English and Odia stop-words (e.g. "heuchi", "laguchi"), and extracts unigrams, bigrams, and trigrams.
  • TF-IDF Weighting: Dampens common terms and amplifies diagnostic markers via Inverse Document Frequency: IDF(t) = ln((1 + N_docs) / (1 + DF(t))) + 1
  • Multinomial Naive Bayes: Calculates posterior classes using Laplace smoothing (α=1): ln P(C|W) = ln P(C) + ∑ IDF(t) * ln P(t|C)
  • Fuzzy Trie Sliding Match: Sliding window scans input character strings in O(L) time. Exact multi-word matches inject an immediate posterior score boost: TrieBoost(C) = ∑ 1.5 * IDF(phrase)
  • Linear SVM: One-vs-Rest Support Vector Machine trained client-side via SGD with hinge loss to establish strict decision boundaries between overlapping symptoms.

Scores are normalized via temperature Softmax (T=2.5) to render clean visual probability bars from 0% to 100%.

⚡ 2. PARALLEL WEBGPU PROGRESSION RUNS [TAP TO EXPAND]

To simulate complex physiological vital trajectories, the system executes 16,384 parallel calculations using WebGPU Shading Language (WGSL) compute shaders:

  • Autonomic Cardiac Progression: HR_next = HR_base + (severity * 8.5) + sin(time * 0.2) * (age / 12) + noise
  • Thermal Hyperpyrexia Shift: Temp_next = Temp_base + (severity * 0.8) + ((heat_index - 70) / 40) * 0.5
  • Oxygen Dissociation Curve: SpO2_next = SpO2_base - (severity * 2.4) - ((HR_next - HR_base) / 15) * 0.3

Calculations write to a 32-entry float buffer, displaying a 1,024-point density scatterplot on the canvas, with a seamless, high-speed JavaScript CPU fallback if WebGPU is unavailable.

💊 3. CLINICAL ICD-11, SNOMED CT & PGX SAFETY [TAP TO EXPAND]

Features a structured offline medical knowledge base containing 17 ICD-11 conditions, 47 SNOMED CT medications, and comprehensive clinical safety override pathways:

  • Allergic Contraindication Overrides:
    • Penicillin Allergy: Intercepts Amoxicillin or Ampicillin and substitutes with **Azithromycin 500mg** (SNOMED CT: `372822002`).
    • NSAID Allergy: Intercepts anti-inflammatories (Aspirin, Ibuprofen, Diclofenac) and substitutes with **Paracetamol 650mg** (SNOMED: `387584000`, Calpol/Crocin) to protect renal & mucosal tracts.
  • Pharmacogenomic (PGx) Metabolic Interceptors:
    • G6PD Deficiency: Blocks hemolytic compounds (such as Nitrofurantoin) and substitutes with safe **Ciprofloxacin 500mg**.
    • HLA-B*5701 Genotype: Intercepts Abacavir to prevent severe systemic hypersensitivity reactions.
    • CYP2D6 Poor Metabolizer: Detects metabolic failures converting prodrug Codeine to active morphine, overriding it with safe analgesics.
  • Vitals Staging: BP, Heart Rate, SpO2, and Temp are compiled. Deviations exceed safety limits to classify patient into Stage 1, Stage 2, or Stage 3 (Critical).

Medications in Chat UI and A4 PDFs are hover-responsive links querying bio-equivalent alternatives instantly upon user click.

🔒 6. AES-256-GCM ENCRYPTED LOCAL SESSION BACKUPS [TAP TO EXPAND]

To guarantee security and privacy of patient information on shared devices, session backups utilize the native browser **Web Crypto API**:

  • PBKDF2 Key Derivation: Derives a secure AES key from user password using a random 16-byte salt and **100,000 iterations** of SHA-256.
  • AES-GCM Encryption: Encrypts the serialized patient profile, chat ledger, and binary radiographs stored in IndexedDB under **AES-256-GCM** with a random 12-byte IV, generating a secure ciphertext with a 16-byte authentication tag to guarantee absolute data integrity.

Incorrect passwords or modified backup structures during restore trigger decryption errors, safely discarding unauthorized entries.

🔬 7. CORPUS INJECTION & CLINICIAN DELTA LEARNING [TAP TO EXPAND]

Submitting a correction triggers an active weight shift: W_correct = W_correct + η * X_symptoms W_predicted = W_predicted - η * X_symptoms where learning rate η = 0.05.
